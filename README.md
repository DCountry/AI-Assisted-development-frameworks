# 🤖 AI 辅助开发框架

> 本仓库汇总了目前开源的 AI 辅助代码开发框架、范式和最佳实践

---

## 📚 框架列表

### 1. [**Superpowers**](https://github.com/dcereijo/superpowers)
为您的编码代理提供完整的软件开发工作流。构建在一组可组合的"技能"和初始指令之上，确保代理正确使用它们。

### 2. [**OpenSpec**](https://github.com/OpenSpec)
将人类和 AI 编码助手与规范驱动开发对齐，在编写任何代码之前，确保您对要构建的内容达成一致。

### 3. [**BMAD-METHOD**](https://github.com/bmad-method/BMAD)
一个由 21 个专门代理、50+ 个引导工作流和自适应智能组成的 AI 驱动敏捷开发框架。它可以从错误修复调整到企业级系统。

### 4. [**SuperClaude Framework**](https://github.com/SuperClaude/SuperClaude)
一个元编程配置框架，通过行为指令注入和组件编排，将 Claude Code 转变为结构化开发平台。提供系统化工作流自动化、强大的工具和智能代理。

### 5. [**Spec-Kit**](https://github.com/spec-kit/spec-kit)
规范驱动开发改变了传统软件开发的方式。几十年来，代码一直是核心——规范只是我们在开始"真正的编码工作"后就抛弃的搭建工具。规范驱动开发改变了这一切：规范变成可执行的，直接生成工作实现，而不仅仅是指导它们。
